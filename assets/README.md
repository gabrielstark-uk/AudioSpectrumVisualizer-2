# Application Assets

This directory contains assets for the desktop application:

- `icon.png` - 512x512 PNG icon for Linux
- `icon.ico` - Icon for Windows
- `icon.icns` - Icon for macOS

You'll need to create these icon files for your application.

## Icon Creation Tools

- For macOS `.icns` files: Use IconUtil or online converters
- For Windows `.ico` files: Use online converters or tools like GIMP
- For Linux, a high-resolution PNG is sufficient

## Icon Requirements

- Windows: 256x256 pixels minimum
- macOS: 1024x1024 pixels recommended
- Linux: 512x512 pixels recommended

Place your icons in this directory before building the application.
